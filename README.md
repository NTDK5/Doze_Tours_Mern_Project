# Dorze Tours Mern Stack Project

This repository contains a full-stack web application built using the **MERN** stack (MongoDB, Express.js, React, and Node.js). The project is divided into two parts:

* **Frontend** : Developed using React and deployed on Vercel.
* **Backend** : Built with Node.js and Express, connected to MongoDB, and deployed on Render.

## Features

### Frontend (React)

* Responsive and dynamic UI built with React.
* Communicates with the backend via RESTful API calls.
* State management using React Hooks.
* Styled with Tailwind CSS for modern and flexible design.
* Handles user authentication and form submissions.

### Backend (Node.js + Express)

* RESTful API with various endpoints for handling client requests.
* MongoDB for database management.
* User authentication and authorization using JWT (JSON Web Tokens).
* File uploads using Multer.
* CORS configured for secure communication with the frontend.

## Usage

Once the installation is complete, you can run both the frontend and backend servers locally.

* Frontend: Runs on `http://localhost:3000/`.
* Backend: Runs on `http://localhost:5000/` (or the port specified in your `.env`).

### Example Endpoints (Backend)

* `GET /api/users`: Fetches all users.
* `POST /api/auth/login`: Logs in a user and returns a token.
* `GET /api/tours`: Fetches all tours.

## Deployment

### Frontend (Vercel)

The frontend is deployed on Vercel. To deploy, follow these steps:

1. Link your GitHub repository to Vercel.
2. Set the root directory to `frontend/`.
3. Set the build command to `npm run build` and output directory to `build`.

### Backend (Render)

The backend is deployed on Render. To deploy:

1. Link your GitHub repository to Render.
2. Set the root directory to `backend/`.
3. Set the build command to `npm install` and the start command to `npm start`.
4. Configure environment variables on Render such as `MONGO_URI` and `JWT_SECRET`.

## Tech Stack

* **Frontend** : React, Tailwind CSS, Axios
* **Backend** : Node.js, Express.js, MongoDB, Mongoose, JWT, Multer
* **Deployment** : Vercel (Frontend), Render (Backend)
